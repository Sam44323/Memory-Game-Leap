# Memory Game Card

**A simple memory game for cards**

### Tech-stacks

- Typescript
- React.JS
- React-Flip
- SCSS

---

### After cloning the repo, run the following commands

---

### Installing the packages

---

```bash
yarn start
```

### Running the dev server

---

```bash
yarn start
```

### Creating the build script

---

```bash
yarn build
```
